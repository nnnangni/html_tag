###### 2. 1요소

HTML의 element는 태그와 내용(contents)로 구성되어 있다.

```html
<h1> 웹문서 </h1>
```



###### 2.2 Self-closint element

닫는 태그가 없는 태그도 존재한다.

```html
<img src = 'url'/>
```



###### 2.3 속성

태그에는 속성이 지정될 수 있다.

```htm
<a href = 'google.com'/>
```

###### 2.4 DOM트리

태그는 중첩되어 사용가능하며, 이 때 다음과 같은 관계를 갖는다.

```html
<body>
    <h1>웹문서</h1>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
    </ul>
</body>

```
